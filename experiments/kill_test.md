# Kill Experiment

Goal: Attempt to invalidate the EED model by constructing counterexamples.

## Method

Compare algorithms that:
- Solve the same problem
- Are engineered to match across ΔS, E, and D

## Result

At small scale (n=5):
- Trivial matches observed (e.g., bubble vs selection)

At larger scales:
- No identical (ΔS, E, D) triplets observed

## Observation

When one or two dimensions align, the remaining dimension diverges.

## Statement

"We tried to pin all three. The system refused."
